XPack

XPack is a simple package manager for arch linux made by python.

How to run xpack?

First make sure you have the following-

  • python 2
  • python2-tk(Tkinter)
  • packer

Next download all the files in master. And last go to the directory with the files in it and type-
sudo python xpack.py And your done!
